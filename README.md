# Secure_Stratup_Vault

A high-security document management system designed for startups. This vault uses **AES-256** and **RSA-2048** encryption to ensure that even if the database is compromised, the files remain unreadable.

## âœ¨ Features
- **End-to-End Encryption:** Files are encrypted on upload and decrypted only during download.
- **Personal Vault:** Private storage for individual user documents.
- **Company Vaults:** Shared storage for teams with Role-Based Access Control (RBAC).
- **Activity Logging:** Full audit trail of file accesses and downloads.
